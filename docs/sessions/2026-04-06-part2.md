## Bugs & Fixes

### 15. Variable name conflict — white screen
**Problem**: `models` variable declared twice in ModelsPage (from matrixData destructuring + useState)
**Solution**: Renamed local state to `chatModels`

### 16. Format badge shows SAFETENSORS for MLX models
**Problem**: Backend format detection missed some MLX models (no `mlx` tag, only library_name)
**Solution**: Added `library_name === 'mlx'` and model ID check (`-mlx` in name). Frontend hardcodes MLX badge.

### 17. Search filters too narrow
**Problem**: Filtering parameters client-side on 50 API results — "qwen" + 397B params = 1 result. But "qwen 397" = 31 results.
**Solution**: Include quant/params filter values in the HF API search query string
